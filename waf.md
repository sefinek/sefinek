# 1
## Bypass cache for CDNs
```
(starts_with(http.request.full_uri, "cdn.")) or (starts_with(http.request.full_uri, "screenshots."))
```

### Eligible for cache
### Edge TTL
#### Ignore cache-control header and use this TTL
### Browser TTL
#### Respect origin TTL
